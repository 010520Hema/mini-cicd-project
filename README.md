# Mini CI/CD Project – Jenkins + Docker

## Project Overview
This project demonstrates a simple CI/CD pipeline integrating GitHub, Jenkins, and Docker.

## Workflow
GitHub → Jenkins → Docker Build → Container Deployment

## Tools Used
- GitHub (Version Control)
- Jenkins (CI/CD Automation)
- Docker (Containerization)

## Features
- Automated Docker image build
- Automatic container deployment
- Pipeline error handling
- Versioned builds

## Outcome
Built a working CI/CD pipeline that automates build and deployment using Jenkins and Docker.
